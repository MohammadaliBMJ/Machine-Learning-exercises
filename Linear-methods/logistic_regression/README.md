## Logistic Regression
This project contains two logistic regression notebooks. One using `sklearn` and the other from scratch.

## Files
- `logistic_regression_sklearn.ipynb`: In this notebook the data is generated using `make_classification` function. Then a Logistic Regression model is trained and different Error measures are calculated for the model. 

- `logistic_regression`: In this file a Logistic Regression model is trained based on the Binary Cross Entropy Loss Function and using Gradient Descent. The weights are initialized to 0.
